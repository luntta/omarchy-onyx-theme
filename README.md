# Onyx

A dark, frosted-glass theme for [Omarchy](https://omarchy.org/) — glossy near-black
charcoal with translucent blurred surfaces, soft white gradient borders, a restrained
pastel terminal palette, watery Hyprland animations, and a consistent 4px radius
throughout.

## Install

```bash
omarchy-theme-install https://github.com/luntta/omarchy-onyx-theme
omarchy theme set Onyx
```
